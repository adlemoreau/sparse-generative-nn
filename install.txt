Clone the repository: 
git clone https://github.com/adlemoreau/sparse-generative-nn.git

Install the marching cubes:
cd sparse-generative-nn/torch/marching_cubes
python setup.py develop
python setup.py install

Install SparseConvNet:
Create folder ARSENAL in sparse-generative-nn/torch to clone SparseConvNet
cd ../ARSENAL
Clone the SparseConvNet repository: git clone https://github.com/facebookresearch/SparseConvNet.git
cd SparseConvNet
bash develop.sh
Check that SparseConvNet is properly installed: cd examples
python hello-world.py

Setup the environment:
Install missing packages: pip install scipy plyfile plotly
cd sparse-generative-nn/torch
Add data (input and target scans) in ./DATA/input_scans (.sdf files) and ./DATA/target_scans (both .sdf and .knw files)
Upload the pretrained model sgnn.pth if needed
